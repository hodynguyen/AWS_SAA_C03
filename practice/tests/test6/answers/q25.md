# Question 25

Airline chatbot for booking, updates, check-ins. Text-based with AI intent analysis and automated actions.

Which is the recommended solution?

## Options

A. Amazon Rekognition + Lambda.

B. Amazon Polly with SSML + Lambda.

C. Amazon Lex + Lambda.

D. Amazon Comprehend + Lambda.

## Correct Answer

**C. Amazon Lex + Lambda.**

## Explanation

Amazon Lex:
- **Conversational AI**: Build chatbots with NLU
- **Intent recognition**: Understand user requests
- **Lambda integration**: Fulfill actions via code hooks
- **Multi-platform**: Deploy on web, mobile, messaging

### Why other options are incorrect:

- **A** - Rekognition is for image/video analysis.

- **B** - Polly is text-to-speech, not chatbots.

- **D** - Comprehend is NLP for insights, not chatbots.
