# Question 59

A company wants to grant temporary access to AWS resources for mobile application users. The users should not need to create AWS accounts.

Which service should be used?

## Options

A. AWS IAM users with temporary credentials.

B. Amazon Cognito identity pools.

C. AWS Directory Service.

D. AWS Single Sign-On.

## Correct Answer

**B. Amazon Cognito identity pools.**

## Explanation

Amazon Cognito identity pools enable you to grant temporary AWS credentials to users, allowing guest access and federation with identity providers. Users don't need AWS accounts.

### Why other options are incorrect:

- **A** - IAM users require AWS account creation.

- **C** - Directory Service is for enterprise directories.

- **D** - SSO is for enterprise workforce.
